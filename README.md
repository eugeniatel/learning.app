# Curriculum

A local deep-study workspace for running multiple subjects in parallel.

The app is a tracker and notebook, not a tutor. It keeps modules, concepts, resources, questions, reviews, and weekly sessions on disk so the curriculum remains inspectable and editable.

## Current Subjects

- LLM and AI systems
- DeFi and protocol risk
- Research and writing craft

## Run

```bash
pnpm dev
```

Open `http://localhost:3000`.

## Verify

```bash
pnpm exec tsc --noEmit
pnpm lint
pnpm build
```
